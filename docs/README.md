## What RailwayBot does

RailwayBot listens for Railway deploy webhooks and turns failures into trackable Discord incidents.

Main features:

- Incident creation and deduplication
- Discord thread-based incident discussion
- Slash commands for triage and routing
- Project muting for noisy services
- Daily usage reports and digests
- Health heartbeat messages
